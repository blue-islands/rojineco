ねこずかんに登録

【このアプリについて】

このアプリは、あなたの街のねこを撮って作る、あなただけのねこずかんです。

それ以外の目的での利用や迷惑行為、犯罪行為には絶対に使用しないでください。

また、このアプリでは、ねこの居やすい場所、ねこの写真を紹介しておりますが、必ず出会えるわけではありませんので、予めご了承ください。

【ねこに出会う時の心構え】

ねこに会いに行くときは、そこの場所のルールに従ってください。

餌やり等の行為は禁止されている場合がほとんどです。また、嫌がるねこを追い回したり、無理やり触ることは虐待と同じです。

「触らしてもらっている」そのような謙虚な心構えでねこと接しましょう。
