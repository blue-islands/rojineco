【このアプリについて】

このアプリは猫に出会いたい人が野良猫に出会うためのアプリです。

それ以外の目的での利用や迷惑行為、犯罪行為には絶対に使用しないでください。

また、このアプリは野良猫がいるスポットを紹介しておりますが、必ず出会えるわけではありませんので、予めご了承ください。

【猫に出会う時の心構え】

猫に会いに行くときは、そこのスポットのルールに従ってください。

餌やり等の行為は禁止されていることが殆どです。また、嫌がる猫を追い回したり、無理やり触ることは虐待と同じです。

「触らしてもらっている」そのような謙虚な心構えで接しましょう。
